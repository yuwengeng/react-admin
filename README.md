# react-guli

1.1. 项目描述/技术选型

1) 此项目为一个前后台分离的后台管理的 SPA, 包括前端 PC 应用和后端应用
2) 包括用户管理 / 商品分类管理 / 商品管理 / 权限管理等功能模块
3) 前端: 使用 React 全家桶 + Antd + Axios + ES6 + Webpack 等技术
4) 后端: 使用 Node + Express + Mongodb 等技术
5) 采用模块化、组件化、工程化的模式开发
